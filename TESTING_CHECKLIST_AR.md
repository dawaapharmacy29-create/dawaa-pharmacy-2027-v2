#### ⚡ استخدم Chrome DevTools
```
1. اذهب إلى: Performance tab
2. انقر على Record
3. قم بالعملية المطلوبة
4. انقر على Stop
5. لاحظ المقاييس
```

```
1. اذهب إلى: Memory tab
2. انقر على Take heap snapshot
3. شغّل العملية
4. خذ snapshot آخر
5. قارن الفرق
```

### اختبارات المستخدم - User Testing

#### 👤 اختبر البيانات الفارغة
```typescript
// الحالة 1: لا توجد بيانات مبيعات على الإطلاق
- يجب عرض رسالة: "لا توجد بيانات مبيعات"
- يجب عدم إظهار الرسوم البيانية الفارغة
- يجب عدم الأخطاء في Console

// الحالة 2: بيانات جزئية
- بعض الأيام بها مبيعات وبعضها لا
- يجب رسم البيانني بشكل صحيح
- يجب الحساب الصحيح

// الحالة 3: تحميل جارٍ
- يجب عرض مؤشر التحميل
- يجب عدم عرض البيانات القديمة
- يجب عدم السماح بالنقر قبل التحميل
```

#### 👤 اختبر التنقل السريع
```
1. انقر على الداشبورد
2. غيّر التاريخ 5 مرات
3. غيّر الفرع 5 مرات
4. لا يجب أن يكون هناك تأخير ملحوظ
5. لا يجب أن تكون هناك أخطاء
```
